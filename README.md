## Zero Shot Classification App using HuggingFace

This app uses
 - streamlit
 - poetry
 - docker

To run - 
1. Clone the repo
2. docker build -t <your-tag> .
3. docker run -it -p 8501:8501 <your-tag>
4. goto - localhost:8501
